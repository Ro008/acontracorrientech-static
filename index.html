El fascinante mundo del desarrollo web frontend 

El desarrollo de páginas web se divide en dos tipos de tecnologías bien diferenciadas: 

El frontend

El escaparate de una web, lo que el usuario puede ver y con lo que puede interactuar. No existe una carrera universitaria centrada en este sector. Un programador frontend sabe lenguajes como HTML, CSS y JavaScript.

El backend 

La trastienda, donde se almacenan pedidos, se gestionan datos del usuario, entre muchos otros aspectos. Los lenguajes del backend se estudian en la carrera de ingeniería informática. Hay cientos de ellos, como Python, Java o PHP.

Existe una gran demanda de programadores de ambos sectores, con la gran diferencia de que para convertirte en desarrollador frontend podrás formarte por tu cuenta, como lo hice yo, y aspirar a muchas ofertas sin necesidad de tener un título universitario relacionado. 

Porque la era de la “titulitis” está llegando a su fin, al menos, en sectores como el tecnológico

 A muchas empresas tecnológicas, incluyendo gigantes como Tesla, les importa bien poco tu título, siempre que demuestres que sabes desenvolverte con los lenguajes de programación requeridos.  

Por todos estos motivos, considero que estás ante una oportunidad que puede cambiarte la vida para mejor. A mí me la ha cambiado.
